# Referenz-Bilder (REPLACE)

Projektbilder hier ablegen:

- `/<slug>/cover.jpg` – Titelfoto für Grid und Detailseite
- `/<slug>/1.jpg`, `/<slug>/2.jpg`, … – Galerie auf der Detailseite

Beispiel: `referenzen/wohnanlage-schwabing/cover.jpg`

Aktuell nutzt die Config Platzhalter (`/placeholder.svg`). Nach dem Eintragen echter Referenzen in `/lib/referenzen/projects.ts` die Pfade auf z. B. `/referenzen/wohnanlage-schwabing/cover.jpg` umstellen.
